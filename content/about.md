## 안녕하세요 👋

저는 **sectoreye**입니다.

백엔드 엔지니어링과 AI 도구 활용에 관심이 많은 개발자입니다.

---

## 관심 분야

- 🏗 **백엔드 아키텍처** — 확장 가능한 시스템 설계와 트레이드오프
- 🤖 **AI / LLM** — LLM 기반 에이전트, 프롬프트 엔지니어링, AI 도구 활용
- ☁️ **인프라** — Docker, Kubernetes, CI/CD 자동화
- 📱 **Android / Mobile** — 앱 개발 경험

---

## 기술 스택

**Languages**: TypeScript, Go, Python, Kotlin  
**Backend**: Node.js, NestJS, FastAPI  
**Infra**: Docker, GitHub Actions, OCI  
**AI Tools**: Claude, Cursor, GitHub Copilot

---

## 이 블로그에 대해

이 블로그는 제가 배우고, 실험하고, 실패한 것들을 기록하는 공간입니다.
